# 美股战术分析 Skill

一个面向大模型的美股分析技能，用于把零散的市场信息整理成可复用的战术分析框架。

它不强调复述新闻结论，而强调回答三个核心问题：

1. 市场当前在交易什么主叙事
2. 真正推动价格的是哪类资金
3. 什么条件会确认延续、导致降速，或使原判断失效

## 适用场景

- 分析标普500、纳指、Russell 2000 与核心科技股
- 拆解地缘政治、油价、通胀、财报与流动性对盘面的影响
- 判断一段走势属于反弹、反转、轧空、融涨还是趋势延续
- 提炼结构化的指数、板块、个股分析流程

## 技能特点

- 先看主叙事，再看边际资金，而不是直接下结论
- 把技术位当作行为切换点，而不是孤立数字
- 强调板块轮动、指数环境与个股结构的联动分析
- 默认使用条件推演，而不是确定性预测

## 输出框架

技能默认按以下顺序组织分析：

1. 市场状态
2. 主叙事
3. 资金分析
4. 指数关键位
5. 板块地图
6. 个股结构
7. 情景树
8. 最终战术判断

## 使用方式

在支持技能调用的环境中，可直接使用：

```text
使用 $us-equity-tactical-analysis 分析当前美股市场状态、边际资金、关键触发位，以及主要指数和核心个股的情景路径。
```

## 定位说明

这是一个偏战术分析的 Skill，适合短中期市场结构研判。  
它不是纯基本面估值工具，也不是自动给出买卖建议的信号系统。
